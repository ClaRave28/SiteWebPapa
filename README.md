## 1. Introduction

Academic portfolio website built for **Pierre Del Castillo**, a mathematics teacher. It centralizes his teaching resources for higher education students (L1 to M1), organized by academic level and subject.

---

## 2. Tools using

- **HTML5** — page structure
- **CSS3** — custom styles (`index.css`)

---

## 3. Stylistics tools using

- **Bootstrap 5.3** — responsive grid, navbar, offcanvas, cards
- **Google Fonts / Material Icons** — Material icon set
- **Custom SVG** — hand-crafted mathematical illustration (open book with symbols ∫ Σ π Δ ∞)
- **Bordeaux/rose color palette** — custom CSS variables (`--cherry-rose`, `--night-bordeaux`, `--lavender-blush`…)

---

## 4. Features list

- Home page with hero section and custom SVG mathematical illustration
- Responsive navbar with **offcanvas** menu on mobile (Bootstrap)
- **Higher Education** page (`Superieur.html`) with cards organized by level (L1, L2, L3, M1)
- Teaching resources organized by subject:
  - L2 — Numerical Analysis
  - L3 — Matrix Analysis
  - M1 — Functional Analysis, Optimization
- Tutorial folders organized by academic year (e.g. `tdamatricielle20242025`, `tdoptimisationannee20242025`, etc.)
- Automatic deployment via **GitHub Actions** (CI/CD workflow)

---

## 5. How to launch ?

No dependencies to install. The website is fully static.

**Locally:**

Simply open `index.html` in a browser, or use the **Live Server** extension (VS Code) for auto-reload:

```bash
# Clone the repository
git clone https://github.com/ClaRave28/SiteWebPapa.git
cd SiteWebPapa

# Open index.html in your browser
```

**Online:**

The site is deployed via GitHub Actions and available at:
> `https://clarave28.github.io/SiteWebPapa/`

## 1. Introduction

Site web portfolio académique réalisé pour **Pierre Del Castillo**, professeur de mathématiques. Il centralise ses ressources pédagogiques destinées aux étudiants du supérieur (L1 à M1), organisées par niveau et par matière.

---

## 2. Technologies utilisées

- **HTML5** — structure des pages
- **CSS3** — styles personnalisés (`index.css`)

---

## 3. Outils stylistiques utilisés

- **Bootstrap 5.3** — grille responsive, navbar, offcanvas, cards
- **Google Fonts / Material Icons** — icônes Material
- **SVG personnalisé** — illustration mathématique (livre ouvert avec symboles ∫ Σ π Δ ∞)
- **Palette bordeaux/rose** — variables CSS personnalisées (`--cherry-rose`, `--night-bordeaux`, `--lavender-blush`…)

---

## 4. Liste des fonctionnalités

- Page d'accueil avec hero section et illustration SVG mathématique
- Navbar responsive avec menu **offcanvas** sur mobile (Bootstrap)
- Page **Enseignement Supérieur** (`Superieur.html`) avec cards organisées par niveau (L1, L2, L3, M1)
- Ressources pédagogiques organisées par matière :
  - L2 — Analyse numérique
  - L3 — Analyse matricielle
  - M1 — Analyse fonctionnelle, Optimisation
- Dossiers de TD organisés par année académique (ex : `tdamatricielle20242025`, `tdoptimisationannee20242025`, etc.)
- Déploiement automatique via **GitHub Actions** (workflow CI/CD)

---

## 5. Comment lancer le projet ?

Aucune dépendance à installer. Le site est entièrement statique.

**En local :**

Il suffit d'ouvrir `index.html` dans un navigateur, ou d'utiliser l'extension **Live Server** (VS Code) pour le rechargement automatique :

```bash
# Cloner le dépôt
git clone https://github.com/ClaRave28/SiteWebPapa.git
cd SiteWebPapa

# Ouvrir index.html dans le navigateur
```

**En ligne :**

Le site est déployé via GitHub Actions et accessible à l'adresse :
> `https://clarave28.github.io/SiteWebPapa/`
